# NuGet Alternatives to SunamoTest

This document lists popular NuGet packages that provide similar functionality to SunamoTest.

## Overview

Testing utilities

## Alternative Packages

### xUnit
- **NuGet**: xunit
- **Purpose**: Modern test framework
- **Key Features**: Theories, fixtures, parallel execution

### NUnit
- **NuGet**: NUnit
- **Purpose**: Mature test framework
- **Key Features**: Rich assertions, test cases, setup/teardown

### MSTest
- **NuGet**: MSTest.TestFramework
- **Purpose**: Microsoft test framework
- **Key Features**: VS integration, data-driven tests

### FluentAssertions
- **NuGet**: FluentAssertions
- **Purpose**: Fluent assertion library
- **Key Features**: Readable assertions, better error messages

### Moq
- **NuGet**: Moq
- **Purpose**: Mocking framework
- **Key Features**: Create test doubles, verify interactions

## Comparison Notes

xUnit is most modern. FluentAssertions for better assertions. Moq for mocking.

## Choosing an Alternative

Consider these alternatives based on your specific needs:
- **xUnit**: Modern test framework
- **NUnit**: Mature test framework
- **MSTest**: Microsoft test framework
